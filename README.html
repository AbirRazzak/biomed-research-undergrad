Biomed Research Graph README

Node Colors

There are two different colors of nodes in the graph. The orange node indicates that the node is a central node, a blue node indicates that the node is not.

Hovering

When hovering over the graph using a cursor, multiple pop-ups can be seen. These pop-ups occur when hovering over a node or a link. When hovering over a node, the node's ID (provided from the Facebook API) will be shown. When hovering over a link, the link's value will be displayed. This value represents the number of likes/comments (depending on which type of graph is selected) between the two nodes.

Statistics

With regards to what the "Central Node Value" is measured with, it is measured according to degree, not distance.

Buttons

The reset button resets the page, in case something is not loading. Also very helpful when coding and updating the page. The clear button just clears the current graph that is displayed. Really has no use as selecting a new graph will automatically clear the old graph by default.
